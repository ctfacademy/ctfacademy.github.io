Enable JavaScript Tutorial

JavaScript is a scripting program language used in CTFA challenges. It must be enabled to ensure functionality of the challenges. Failure to enable JavaScript may cause web content to be limited or unavailable. The following guide will help users enable JavaScript in the browsers CTFA supports.

Chrome:

  1. On Chrome web browser window, click on the three dots on the top right of the screen
  2. Select settings from the drop-down menu
  3. At the bottom of the page click on Advanced
  4. Expand Site Settings by clicking the right arrow to the right of ‘Site Settings’
  5. Ensure JavaScript says Allowed. If not click and change to allowed

Opera:

  1. Click Opera icon on top left of Opera web browser
  2. Select settings from the drop-down menu
  3. Click on Advanced
  4. Select Privacy and security
  5. Select content settings
  6. Ensure JavaScript says Allowed. If not click and change to allowed

Firefox:

  1. Open Firefox web browser and type about:config in the address bar
  2. Click accept the risk
  3. Type javascript.enabled in the search box
  4. Ensure that the value column is true. If it isn’t, change to true by double clicking false and change value to true

Edge:

Microsoft Edge Home and Starter Editions are preconfigured to allow JavaScript and cannot be changed.

Internet Explorer:

  1. Click setting icon on top right of Internet Explorer (IE) web page
  2. Select Internet Options from pop-up window
  3. Go to Security tab then click custom level
  4. Scroll to scripting and ensure active scripting is enabled and if not click enabled then ok
