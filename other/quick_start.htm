CTF Academy Quick Start Guide

Welcome to CTF Academy, making cyber security fun and exciting for everyone. CTF Academy is a curriculum aimed at high school level student and above.

To begin teaching this exciting curriculum, follow these simple steps.

  1. First, download the “ctfacademy.zip” file to your local system.
  2. Next, right-click on the zip file and select “Extract All…” from the context menu.
  1. You should now be prompted with a new window asking you to select where you want the file to be extracted. Click “Browse…” if you wish to change the extraction location; otherwise, click “Extract.”
  1. Next, navigate to the newly created “ctfacademy” folder and open it. You should be presented with the following list of files and folders.
  1. Now, right-click on the “index.htm” file, hover over “Open with” and then select your preferred web browser from the context menu. Google Chrome is the recommended browser for CTF Academy.
  1. Sometimes, you may be prompted with the following window (Windows 10) or a similar window. Select your desired web browser and check the “Always use this to open .htm files” checkbox, so that all other htm files will be opened with a web browser.
  1. After opening the “index.htm” file with a web browser, you should be presented with the home page of CTF Academy as shown below. Simply follow the instructions on the home page to navigate the website and teach students, or yourself, the basics of cyber security.
